# Borders - radius and width

Almost all components are slightly rounded. Our radius system follows a rule of multiple of 2.<br />
Scale creates a visual balance that makes it easier to understand the interface.

Border width is primarily used to outline the frame of a component or to structure content.<br />
We use an inset border width of 1 px in form elements and shadow buttons.
